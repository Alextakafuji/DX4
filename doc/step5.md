# Step 5 クリアとゲームオーバーを実現しよう

`cell.innerText`でそのマスの0〜9の値を取得することができますので、
このメソッドを改造してマスが押された時の動作を実装してみましょう。

爆弾の設置されているマスがクリックされたときに、ゲームオーバーになるようにしてみましょう。
Step4では`handle_table_click(cell)`でCSSを切り替えることによって、マスの表示を変えていました。

### ヒント
```js
if (cell.innerText === `9`) {
	// 爆弾が押された！
	} else {
	// セーフ！
	}
```

## Q2.ゲームクリア

### 解説
全てのマスが押されたか、押されていないかを検査する必要がありますので、`table`全体を見る必要がありそうです。
Q1で修正したメソッドに引数を追加して`table`を参照できるようにしましょう。

### やってみよう
爆弾のないマスが全て押されたらゲームクリアになります。
ゲームクリアの処理を実装してみましょう。

### ヒント
マスが押されたか押されていないかは`cell.className`で確認することができます。

```js
if (cell.className === "cell off") {
	// 押されていない
	} else {
	// 押されている
	}
```

全ての爆弾のマスは押されていないはずですので、どのようなIf文になるのか考えてみましょう。
